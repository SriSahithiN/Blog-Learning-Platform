# 🚀 LearnDev — Interactive Web Development Learning Platform

A modern, interactive blog-style learning platform for mastering Web Development technologies like HTML, JavaScript, and React.

🔗 **Live Demo:** https://your-vercel-url.vercel.app

---

## ✨ Features

* 📚 Structured Courses (HTML, JS, React)
* 🧠 Step-by-step Lessons
* 💻 Live Code Editor with Test Cases
* ✅ Progress Tracking System
* 💡 Hint System for Learning
* 🔍 Smart Search (Lesson Navigation)
* 🌙 Dark Mode Support
* 📱 Fully Responsive Design
* ⚡ Smooth Animations (Framer Motion)

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* React Router
* Context API (State Management)
* LocalStorage (Persistence)
* Framer Motion (Animations)
